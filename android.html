The offical app for SHS countdown is here! Finally you can know what time it is without taking out a computer or accessing the internet.
There will not be an iOS version due to lack of an apple account and a mac. The download button is at the bottom.

Features:
> Automatic exception handling with server data sync
> Manual time syncing
> Custom day setting option
> Schedule and time data for the day
> Countdown for class, school, summer, lab and school days.

Installation instructions:
1. Make sure you have allowed installation from unknown sources (Settings > security)
2. Download the apk from the link below onto your device.
3. Start up a file explorer, navigate to your download directory and click on the APK
4. Accept permissions and install.




Device recommendations: phone. Interface may break on 3:4 screen ratios. See installation instructions.
